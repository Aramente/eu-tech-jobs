<p><strong>POSITION SUMMARY</strong></p>
<p>The HR Technology Analyst will support the design, maintenance, and enhancement of our HR technology ecosystem, with an emphasis on <strong>Workday integrations</strong>, <strong>ServiceNow HRSD</strong>, and <strong>HR data pipelines</strong>.</p>
<p><strong>ROLES AND RESPONSIBILITIES</strong></p>
<p>Workday &amp; ServiceNow Integration Support</p>
<ul>
<li>Assist in building and maintaining integrations between <strong>Workday</strong> and internal/external systems using <strong>EIBs, Core Connectors, REST/SOAP APIs</strong>, and file-based exchanges.</li>
<li>Support the <strong>implementation and configuration of ServiceNow HRSD</strong>, including workflows, case management, knowledge articles, and integrations with Workday.</li>
<li>Monitor integration jobs, resolve errors, and perform root-cause analysis with senior analysts or architects.</li>
<li>Participate in code reviews, data mapping exercises, and testing cycles for new integrations.</li>
</ul>
<p>Technical Implementation &amp; Data Processes</p>
<ul>
<li>Participate in data mapping and transformation during HRSD implementation and post-go-live enhancement cycles.</li>
<li>Support automation initiatives using integration tools or scripts to reduce manual effort and improve HR data flow reliability.</li>
<li>Contribute to maintaining metadata, documentation, and configuration repositories for integrations.</li>
</ul>
<p>Cross-System Collaboration</p>
<ul>
<li>Work closely with HRIS, Technology, and Security teams to align HRSD and Workday integrations with enterprise architecture and compliance standards.</li>
<li>Collaborate with vendors and implementation partners during configuration, testing, and deployment phases.</li>
<li>Maintain detailed documentation for integrations, data flow diagrams, and configuration changes.</li>
</ul>
<p>Collaboration &amp; Learning</p>
<ul>
<li>Stay up to date on <strong>Workday and ServiceNow</strong> release updates, new features, and best practices.</li>
<li>Participate in technical design sessions and translate business needs into system requirements.</li>
</ul>
<p>&nbsp;</p>
<p><strong>QUALIFICATIONS</strong></p>
<ul>
<li>Bachelor’s degree in Computer Science, Information Systems, or related field (or equivalent practical experience).</li>
<li>2–5 years of experience in HR systems support, data integration, or software development, preferably involving Workday and ServiceNow HRSD.</li>
<li>Basic proficiency with <strong>APIs, XML/JSON,</strong> or <strong>file transfer protocols.</strong></li>
<li>Understanding of <strong>Workday HCM</strong> data structures and integration concepts.</li>
<li>Exposure to <strong>ServiceNow HRSD implementation activities</strong> such as configuration, case management, or workflow setup.</li>
<li>Strong analytical and debugging skills, and a drive to learn technical tools deeply.</li>
</ul>
<p><strong>PREFERRED QUALIFICATIONS</strong></p>
<ul>
<li>Exposure to <strong>Workday Studio</strong> or scripting (Python, PowerShell, or SQL).</li>
<li>Knowledge of data governance, HR security models, or data warehouse integration.</li>
</ul>
