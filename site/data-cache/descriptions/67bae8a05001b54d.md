<h2><strong>The role</strong></h2>
<div>The Production Engineer - Software Integration ensures software runs reliably in live environments, focusing on stability, efficiency, and scalability by building automation, monitoring tools (observability), managing incidents, optimizing CI/CD pipelines, and collaborating cross-functionally to improve developer experience and product quality, bridging DevOps and Software Engineering. Key duties include troubleshooting, performance tuning, deployment automation, incident response, and security, ensuring seamless operations from code to production.</div>
