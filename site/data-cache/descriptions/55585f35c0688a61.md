<p>As a <strong>Network Automation Engineer</strong>, you will design, develop, and implement automation solutions to improve network efficiency, scalability, and reliability. This role is a blend of deep networking expertise and software development, aimed at streamlining network operations and reducing manual intervention.&nbsp;</p>
<p><strong>The impact you will have: </strong></p>
<ul>
<li><strong>Automation Frameworks:</strong> Design and implement network automation frameworks and solutions.</li>
<li><strong>Scripting &amp; Development:</strong> Develop automation scripts using <strong>Python, Ansible, Terraform</strong>, Pulumi, or similar tools.</li>
<li><strong>Provisioning &amp; Deployment:</strong> Automate network provisioning, configuration, and deployment processes.</li>
<li><strong>Monitoring &amp; Remediation:</strong> Monitor network performance and build automated alerting and remediation systems.</li>
<li><strong>CI/CD Excellence:</strong> Architect deployment pipelines using GitHub Actions, managing both GitHub-hosted and self-hosted runners for specialized build requirements.</li>
<li><strong>Integration:</strong> Work with APIs, SDKs, and SDN controllers to integrate automation into the broader infrastructure.</li>
<li><strong>Collaboration:</strong> Partner with Network, DevOps, and Security teams to ensure seamless workflows and compliance with security policies.</li>
<li><strong>Optimization:</strong> Troubleshoot automation failures, optimize existing workflows, and maintain high standards for documentation.</li>
</ul>
<h4><strong>What we are looking for:</strong></h4>
<p><strong>3+ years</strong> of relevant experience as a Network Engineer</p>
<p><strong>Networking Fundamentals:</strong></p>
<ul>
<li>Strong understanding of TCP/IP, DNS, and DHCP.</li>
<li>Expertise in Switching, VLANs, WAN/LAN, and SD-WAN/VPN.</li>
<li>Hands-on experience with routers, switches, and firewalls.</li>
</ul>
<p><strong>Automation &amp; Programming:</strong></p>
<ul>
<li><strong>Python:</strong> Must-have proficiency for scripting and tool development.</li>
<li><strong>IaC Tools:</strong> Experience with Ansible, Terraform, or Pulumi.</li>
<li><strong>Modern Practices:</strong> Proficiency with REST APIs, JSON/YAML, Git, and CI/CD pipelines.</li>
</ul>
<p><strong>Modern Infrastructure:</strong></p>
<ul>
<li>Cloud networking experience (AWS, Azure, or GCP).</li>
<li>Understanding of SDN/NFV concepts.</li>
<li>Knowledge of container networking (Docker/Kubernetes) is a plus.</li>
</ul>
