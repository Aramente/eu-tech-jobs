<p><span style="font-size: 10pt;"><span data-contrast="auto">As the Hardware System Test Engineering Manager, you will lead the Hardware System Test Team responsible for developing manufacturing test solutions for High Performance Modules (HPM), Server Blades, and Multi-node racks as part of the production assembly process. You will guide the development of manufacturing test content&nbsp;and test fixtures&nbsp;informed by&nbsp;a&nbsp;deep understanding of silicon, board, and system design.
