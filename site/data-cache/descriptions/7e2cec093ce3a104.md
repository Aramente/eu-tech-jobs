<h3 style="text-align: justify;"><strong>About the role</strong></h3>
<p><strong></strong></p>
<p class="font-claude-response-body break-words whitespace-normal leading-[1.7]">As a Senior Software Engineer on the Platform team, you'll architect the systems that power PIE - unifying siloed data sources across CRMs, lending systems, dialers, and communication channels into a unified intelligence layer. You're building infrastructure that doesn't just handle data at scale, but extracts meaning from it, enabling AI agents to make contextually intelligent, compliant decisions in real-time.</p>
<p class="font-claude-response-body break-words whitespace-normal leading-[1.7]">This isn't incremental improvement - it's foundational work toward an agentic future where specialized AI agents collaborate autonomously to manage entire collections operations. The systems you build today become the backbone of tomorrow's agentic workforce.</p>

<h3 style="text-align: justify;"><strong>What you'll do</strong></h3>
<p class="font-claude-response-body break-words whitespace-normal leading-[1.7]"></p>
<p><strong></strong></p>
<ul style="text-align: justify;">
<li>Design and build the distributed systems infrastructure that powers PIE, processing millions of consumer interactions daily while maintaining sub-100ms response times for real-time agent decision-making</li>
<li>Build robust data unification pipelines that reconcile and synthesize information across heterogeneous systems (CRMs, LMS platforms, dialers, communication channels) into coherent, actionable intelligence</li>
<li>Create scalable platforms that can ingest, process, and serve insights from 500+ consumer attributes while supporting complex regulatory and compliance requirements</li>
<li>Own end-to-end architecture and implementation - from high-level system design through production deployment and operational excellence</li>
<li>Write production-grade code in Python, Golang, and Node.js that balances performance, maintainability, and speed of iteration</li>
<li>Conduct rigorous code reviews, tackle complex distributed systems bugs, and take ownership of high-risk technical challenges</li>
<li>Build comprehensive testing strategies spanning unit, integration, and regression capabilities to ensure reliability at scale</li>
<li>Implement sophisticated monitoring, observability, and troubleshooting capabilities for complex distributed systems</li>
<li>Drive improvements in key technical metrics: system availability, data freshness, query performance, and cost efficiency</li>
<li>Partner with ML teams to build infrastructure that supports model training, inference, and continuous learning loops</li>
<li>Influence technical direction and architectural decisions that span multiple teams and projects</li>
<li>Mentor junior engineers through code reviews, architectural discussions, and collaborative problem-solving</li>
<li>Champion engineering best practices, documentation standards, and operational excellence across the platform team</li>
</ul>
