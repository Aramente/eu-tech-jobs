<p>Make Wayve the experience that defines your career!&nbsp;&nbsp;</p></div><h2><strong>The role&nbsp;</strong></h2>
<p>Join Wayve to build the in‑vehicle interface that helps people <em>see and understand</em> what an autonomous vehicle is doing in real time. You will turn complex AI outputs, camera feeds, and vehicle state into clear, trustworthy 2D and 3D visualisations that are used in moving vehicles every day. This role spans rapid iteration with real operators and the engineering depth needed for reliable, high-performance multi-platform systems. The impact is direct: you will shape how humans interact with autonomy and how the organisation diagnoses and improves on-road behaviour.</p>
<p><strong>Key responsibilities:</strong></p>
<ul>
<li>Build and evolve an in‑vehicle web application that visualises camera feeds, vehicle state, and system outputs for live operation, demos, and diagnostics.</li>
<li>Design and deliver 3D spatial visualisations of the vehicle’s surroundings (trajectories, perception meshes, occupancy, overlays) using WebGL / Three.js / similar.</li>
<li>Integrate high-frequency real-time data streams and backend services into a responsive, performant UI that runs reliably in constrained environments.</li>
<li>Partner closely with Vehicle Safety Operators and engineers to understand workflows, iterate quickly based on field feedback, and improve usability and operational efficiency.</li>
<li>Own technical quality and long-term maintainability: architecture, testing, observability, performance tuning, and clear end-to-end feature ownership across the stack.</li>
</ul>
<h2><strong>About you</strong>&nbsp;&nbsp;</h2>
<p>In order to set you up for success as a Fullstack Software Engineer at Wayve, we’re looking for the following skills and experience.&nbsp;&nbsp;</p>
<p><strong>Essential</strong></p>
<ul>
<li>Strong JavaScript/TypeScript + React experience building production-grade web applications end-to-end.</li>
<li>Experience building browser-based 3D visualisations (WebGL, Three.js, React Three Fiber, or similar).</li>
<li>Proven ability to integrate frontends with backend services and real-time, high-frequency data streams.</li>
<li>Strong engineering fundamentals around performance, reliability, testing, and maintainable system design, with clear ownership and high autonomy.</li>
<li>Ability to work closely with users and stakeholders (operators and engineers), iterate quickly based on feedback, and still ship robust solutions.</li>
<li>Strong debugging and problem-solving skills, including diagnosing issues across UI, data pipelines, and runtime/perf bottlenecks.</li>
</ul>
<p><strong>Desirable</strong></p>
<ul>
<li>Experience shipping software in constrained or safety-critical environments (for example, in-vehicle, edge, intermittent connectivity).</li>
<li>Experience working across multi-platform systems (web + mobile/Android) and designing cohesive architectures across them.</li>
<li>Demonstrated technical leadership: setting direction, mentoring, and driving quality improvements (observability, scalability, robustness).</li>
<li>Experience with UI performance and rendering optimisation, especially for complex scenes or data-heavy visualisations.</li>
<li>Familiarity with streaming architectures, pub/sub, or telemetry systems that support real-time visualisation and monitoring.</li>
</ul>
