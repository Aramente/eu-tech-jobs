<ul>

<li style="font-size: 10pt;"><span style="font-size: 10pt;">Design and implement intuitive user interfaces to collect and review information.</span></li>
<li style="font-size: 10pt;"><span style="font-size: 10pt;">Build and maintain robust APIs that automate decision-making processes.</span></li>
</ul>
<li style="font-size: 10pt;"><span style="font-size: 10pt;"><strong>Simplify the Complex:</strong> Design, write and test (from unit to end-to-end) clear, clean, well-structured <strong>C# and React</strong> code that is easy for others to maintain.&nbsp;</span></li>
<li style="font-size: 10pt;"><span style="font-size: 10pt;"><strong>Test, Learn, Scale:</strong>&nbsp;</span></li>

<p>We are looking for a Shifter who is:</p>
