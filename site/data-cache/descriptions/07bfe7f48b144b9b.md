<p><span style="text-decoration: underline;"><strong>Role Description:</strong></span></p>
<p>We are seeking a highly motivated Security Operations Center (SOC) Analyst to join our&nbsp;security team. The SOC Analyst will be responsible for monitoring, detecting, analyzing,&nbsp;and responding to cybersecurity threats and incidents. The ideal candidate should have&nbsp;a strong understanding of security tools, incident response procedures, and threat&nbsp;intelligence to safeguard our organization’s digital assets.<br><br></p>
<p><span style="text-decoration: underline;"><strong>Key Responsibilities:</strong></span></p>
<p><strong>Security Monitoring &amp; Incident Response:</strong></p>
<ul>
<li>Monitor security events and alerts in real-time using SIEM and other security tools.</li>
<li>Analyze security incidents, determine root causes, and escalate as needed.</li>
<li>Respond to security incidents, perform forensic analysis, and document findings.</li>
<li>Assist in containment, eradication, and recovery efforts following security breaches.</li>
<li>Maintain incident response documentation and playbooks.</li>
</ul>
<p><br><span style="text-decoration: underline;"><strong>Threat Analysis &amp; Intelligence:</strong></span></p>
<ul>
<li>Investigate security threats, vulnerabilities, and suspicious activities.</li>
<li>Correlate threat intelligence feeds with internal security events.</li>
<li>Identify emerging threats and recommend proactive measures.</li>
</ul>
<p><br><span style="text-decoration: underline;"><strong>Security Tools &amp; Technology:</strong></span></p>
<ul>
<li>Operate and fine-tune security technologies such as firewalls, IDS/IPS, EDR, and DLP solutions.</li>
<li>Support the configuration and maintenance of security monitoring tools.</li>
<li>Assist in tuning detection rules to reduce false positives and improve response efficiency.</li>
</ul>
<p>&nbsp;</p>
<p><span style="text-decoration: underline;"><strong>Technical Skills:</strong></span></p>
<ul>
<li>Strong knowledge of security monitoring tools (SIEM, EDR, IDS/IPS, Firewalls).</li>
<li>Familiarity with threat intelligence platforms and attack frameworks (MITRE ATT&amp;CK).</li>
<li>Experience with forensic analysis, malware detection, and incident response.</li>
<li>Understanding of networking protocols, cloud security, and system administration.</li>
<li>Proficiency in scripting (Python, PowerShell, Bash) is a plus.</li>
</ul>
<p>&nbsp;</p>
<p><span style="text-decoration: underline;"><strong>Certifications (Preferred but Not Required):</strong></span></p>

</ul>
