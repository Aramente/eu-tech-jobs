**At NordLayer, we’re building cybersecurity that scales with the business.**

A toggle-ready platform that helps modern teams thrive—without the security headaches. Trusted by 11,000+ global companies, NordLayer plugs into any tech stack and protects users across borders.

**Your impact?** Helping businesses stay protected and moving forward with future-ready network security.

## Main Responsibilities

- Expand and improve our codebase by writing clean, quality TypeScript code supported by tests.
- Provide technical leadership to frontend developers through code reviews, and guidance on implementation approaches.
- Collaborate closely with backend engineers to define API contracts and ensure reliable integration between frontend and backend services.
- Work with product managers and designers to translate product requirements and UX designs into practical technical solutions.
- Ensure high engineering standards by promoting best practices in code quality, testing, and maintainability.
- Support the team in delivering features in a Scrum environment by helping break down work, identifying risks early, and contributing to estimations.

## Core Requirements

- 5+ years of experience bringing real business value through building and maintaining modern frontend applications using React and TypeScript.
- Experience designing frontend architectures and establishing scalable components and state management patterns.
- Strong understanding of frontend performance optimization and web application best practices.
- Experience working with REST APIs and collaborating closely with backend teams.
- Familiarity with frontend testing frameworks such as Jest, Testing Library, Cypress, or Playwright.
- Experience leading technical discussions within a distributed team environment.

## Bonus Points For

- Hands-on work in building or contributing to shared design systems.
- Experience supporting or leading technical initiatives across multiple teams.
- Knowledge of CI/CD pipelines and modern development workflows.
