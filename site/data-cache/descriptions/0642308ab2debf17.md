<p><strong>About the Role:</strong></p>
<p>We are looking for an engineer with a passion for systems programming and a curiosity for how databases work under the hood. You will be part of a team building the core of our database engine—working on change data capture, connection management, storage engine internals, and performance optimizations.</p>
<p>This is a hands-on role where you'll work deep in C/C++ code, exploring distributed systems and engineering the building blocks that make databases scale efficiently.</p>
<p><strong>Responsibilities:</strong></p>
<ul>
<li>Design, develop, and optimize core components of the database engine (storage layer, change data capture, connection manager, query layer etc.)</li>
<li>Investigate and fix complex issues related to performance, memory usage, and concurrency</li>
<li>Write high-quality, well-tested, and maintainable C/C++ code</li>
<li>Collaborate with senior engineers to understand and extend database internals</li>
<li>Use profiling and debugging tools to diagnose bottlenecks in low-level code paths</li>
</ul>
<p><strong>Requirements:</strong></p>
<ul>
<li>10+ years of experience in systems programming (C/C++ preferred)</li>
<li>Strong understanding of operating systems concepts (processes, threads, virtual memory, synchronization)</li>
<li>Solid knowledge of data structures, algorithms, and computer architecture</li>
<li>Experience debugging and profiling low-level code (e.g., gdb, perf)</li>
<li>Familiarity with networking, storage I/O, or multithreaded programming</li>
</ul>
<p><strong>Nice to have:</strong></p>
<ul>
<li>Exposure to database internals (query planning, indexing, storage engines, transaction processing)</li>
<li>Experience with open-source database projects (PostgreSQL, MySQL, RocksDB, etc.)</li>
<li>Knowledge of distributed systems or consensus protocols (Raft, Paxos)</li>
</ul>
<p><strong>Why Join Us?</strong></p>
<ul>
<li>Work on cutting-edge database technology used in production environments</li>
<li>Solve deep technical problems that require creativity and rigor</li>
<li>Be part of a small, highly skilled team where your work has a visible impact</li>
<li>Opportunity to learn from senior engineers with expertise in databases and distributed systems</li>
</ul>
