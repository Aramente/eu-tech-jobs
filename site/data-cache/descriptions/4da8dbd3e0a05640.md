<h3><strong>The role</strong></h3>
<p>We are looking for an Early Talent to develop cloud-based integrations on Microsoft Azure. You will work with experienced engineers and gain hands-on experience building API-based and cloud-native integration solutions.</p>
<p><strong>We expect you to have</strong></p>
<ul>
<li>0–2 years of experience with Microsoft Azure (study projects, internships, or junior roles count).</li>
<li>Basic understanding of <strong>cloud concepts</strong> (compute, storage, networking).</li>
<li>Familiarity with <strong>Infrastructure as Code</strong> (Terraform / Bicep / ARM) — even at learning or pet-project level.</li>
<li>Basic experience with <strong>CI/CD concepts</strong> (e.g., GitHub Actions, Azure DevOps, or similar).</li>
<li>Understanding of:<br>&nbsp;- Azure Networking basics (VNet, NSG, public/private access)<br>&nbsp;- Compute basics (VMs, containers)<br>&nbsp;- Storage services</li>
<li>Awareness of identity concepts (RBAC, basics of Microsoft Entra ID).</li>
<li>Basic scripting skills (PowerShell / Bash / Python — even simple scripts).</li>
<li>Interest in monitoring and logging (Azure Monitor, Log Analytics — basic exposure is enough).</li>
<li>Understanding of <strong>cloud fundamentals, security basics, and cost awareness</strong>.</li>
<li>English: Intermediate.</li>
</ul>
