We help people and businesses take back control of their online security, privacy, and data. From VPNs to password managers, threat intelligence to eSIMs for travel—our teams turn complex problems into solutions trusted by millions worldwide.

Life is online. In this role, you’ll help people own it.

## Main Responsibilities

- Build and continuously improve a sophisticated User Control Panel that drives user trust, engagement, and recurring growth
- Implement new features and user flows across the account management portal (e.g., subscriptions, billing, security)
- Develop and maintain a high-quality, scalable front-end codebase
- Continuously improve code quality, performance, and overall user experience
- Write and maintain unit and integration tests alongside feature development
- Contribute to and follow component library and design system conventions
- Participate in code reviews and uphold high engineering standards
- Collaborate closely with backend engineers on API contracts and error handling
- Work with QA engineers to ensure reliable feature behavior, including edge cases and documented flows
- Debug and resolve issues across the frontend stack, from UI rendering to state management (e.g., RTK Query)
- Contribute to improving engineering standards, tooling, and development processes
- Conduct R&D activities when needed to support new initiatives or technical decisions
- Provide technical guidance and mentor team members on best practices
- Contribute to the team’s technical vision and long-term goals

## Core Requirements

- 4+ years of experience with JavaScript and TypeScript
- 2+ years of experience with React
- Strong knowledge of web development, testing, and performance optimization best practices
- Proven experience building high-quality, high-performance front-end applications at scale
- Comfortable working in large, mature codebases with many components
- Ability to follow established patterns without introducing unnecessary architectural changes
- Experience collaborating with product and design teams, translating Figma specifications into pixel-perfect, accessible components
- Strong communication skills, with the ability to present ideas, build consensus, and communicate progress clearly
- Experience contributing to a strong code review culture, including giving and receiving constructive feedback
- Proactive in communicating blockers, technical debt, and trade-offs
- Team-oriented, proactive, and comfortable in a fast-paced environment
- Self-motivated and well-organized, with the ability to manage work independently
