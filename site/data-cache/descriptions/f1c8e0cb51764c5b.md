<h1><strong>Kotlin Expert (SME) – Consultant Description</strong></h1>
<h2><strong>Purpose</strong></h2>
<p>We are seeking a highly experienced Kotlin Software Engineering professional to serve as a Subject Matter Expert (SME) on advanced software development and AI benchmarking projects. This role is pivotal in designing, documenting, and validating Kotlin-based tasks and functionalities that test and refine the coding capabilities of large language models (LLMs) — particularly within Android development environments.<br><br>Your work will define both the technical standard and the functional scaffolding of real-world coding benchmarks, ensuring all tasks meet the highest levels of authenticity, precision, and reproducibility. You will help assemble datasets and functional frameworks that mirror the complexity and reasoning required in professional Android engineering.</p>
<h2><strong>Core Components</strong></h2>
<h3><strong>1. Functional Design, Documentation &amp; Benchmark Definition</strong></h3>
<ul>
<li>Define, document, and validate Kotlin-based Android functionalities used in AI benchmarking and software simulation projects.<br>• Author comprehensive functional and technical documentation, including feature requirements, system behavior, and expected outcomes.<br>• Translate open-ended requirements into realistic, well-scoped Android tasks representing genuine engineering challenges.<br>• Ensure each benchmark task is complete, verifiable, and grounded in authentic developer workflows.<br>• Maintain structured repositories of Kotlin challenges, categorized by complexity, topic, and functional type.</li>
</ul>
<h3><strong>2. Technical Standard Setting, QA &amp; Process Improvement</strong></h3>
<ul>
<li>Define Kotlin-specific quality metrics, benchmarking criteria, and rubrics aligned with Android production standards.<br>• Develop SOPs and QA frameworks enforcing stylistic consistency, idiomatic Kotlin use, and functional clarity.<br>• Conduct rigorous QA passes on deliverables — verifying code correctness, documentation completeness, and benchmark adherence.<br>• Apply quality principles to maintain dataset and code task evaluation standards.<br>• Curate gold-standard Kotlin examples representing benchmark-worthy implementations.</li>
</ul>
<h3><strong>3. Talent Definition, Vetting &amp; Calibration</strong></h3>
<ul>
<li>Collaborate with recruitment and project leads to define role competencies for Kotlin contributors.<br>• Develop technical evaluation frameworks for vetting Kotlin developers, including practical code tasks and debugging exercises.<br>• Lead calibration sessions with contractors to ensure consistency and quality.<br>• Create skill matrices and capability documentation for Kotlin engineering talent aligned to benchmarking needs.<br>• Provide continuous feedback and training materials for quality improvement.</li>
</ul>
<h3><strong>4. Benchmark Dataset Development &amp; Validation</strong></h3>
<ul>
<li>Collaborate with AI research teams to design and refine Android benchmarking datasets for evaluating LLM coding performance.<br>• Source, assess, and refine Kotlin-based Android tasks testing comprehension, reasoning, and problem-solving.<br>• Document evaluation criteria including accuracy, code structure, and maintainability.<br>• Contribute to post-project analysis, surfacing insights on performance trends and task design improvements.<br>• Recommend dataset refinements to enhance domain coverage.</li>
</ul>
<h3><strong>5. Project Delivery &amp; Stakeholder Support</strong></h3>
<ul>
<li>Support project setup, scoping, and execution of Kotlin-oriented AI training and evaluation projects.<br>• Provide expert-level input on tooling, frameworks, and pipeline design (Ktor, Jetpack, Gradle, KSP).<br>• Collaborate closely with ML engineers, QA specialists, and data annotators to meet research-grade standards.<br>• Produce summary documentation and retrospective reports detailing lessons learned and recommendations.</li>
</ul>
<h2><strong>Target Profile</strong></h2>
<ul>
<li>5+ years of professional experience in Kotlin software engineering (Android, backend, or multiplatform).<br>• Mastery of Kotlin idioms, coroutines, Flow, Jetpack Compose, and asynchronous programming.<br>• Experience in technical documentation, benchmark dataset design, or QA of coding tasks preferred.<br>• Proven ability to define and enforce technical and stylistic standards across distributed teams.<br>• Strong collaboration and communication skills with the ability to articulate complex logic clearly.<br>• Exceptional attention to detail and documentation discipline.<br>• Fluent in English, with precise writing and communication abilities.</li>
</ul>
