<p>&nbsp;</p></div><p>The security team is a critical component of Shift Technology as no organization is immune to cyber-crime. The team is responsible for protecting information throughout the security infrastructure, edge devices, networks, and data. We strive to stay up to date with the latest tactics hackers are employing in the field in order to prevent data breaches by monitoring and reacting to attacks but the first step is finding the most qualified professionals to lead the way.&nbsp;</p>
<p>&nbsp;</p>
<p><strong>What you'll do...</strong></p>
<p>As a Senior Security Architect, reporting to the CISO, you will be a senior member of the Information Security team, responsible for the end-to-end security architecture of our Azure-based SaaS platform and the creation, adoption, and governance of security architecture best practice across our organisation. You will serve as the subject matter expert for both cloud infrastructure and application security, partnering with engineering teams to embed security into the entire software development lifecycle (SDLC). You will design secure, scalable, and resilient solutions for our single- and multi-tenant offerings, ensuring the protection of our platform and our customers' data.&nbsp;</p>
<p>&nbsp;</p>
<p><strong>RESPONSIBILITIES</strong></p>
<p style="padding-left: 40px;"><strong>Cloud Security Architecture</strong></p>
<ul>
<li>
<ul>
<li>Design and maintain the security reference architecture for our Azure-native, Windows, and Kubernetes-based SaaS products.</li>
<li>Act as the primary security consultant for product and engineering teams, providing authoritative guidance on secure design patterns for Azure infrastructure and services.</li>
<li>Review and approve architectural designs for new services to ensure they align with security principles (Zero Trust, defense-in-depth) and compliance requirements.</li>
<li>Define and enforce security standards for Azure networking, including VNet segmentation, firewalling, and private connectivity.</li>
</ul>
</li>
</ul>
<p style="padding-left: 40px;"><strong>Product &amp; Application Security</strong></p>
<ul>
<li>
<ul>
<li>Lead threat modeling exercises (e.g., STRIDE) with development teams for new products and features to identify and mitigate risks early in the SDLC.</li>
<li>Develop and maintain secure coding standards and provide expert guidance on the prioritization and remediation of findings from SAST, DAST, and SCA tools.</li>
<li>Architect security solutions for the SaaS application layer, including tenant isolation, customer data segregation, secure APIs, and authentication/authorization patterns.</li>
</ul>
</li>
</ul>
<p style="padding-left: 40px;"><strong>Data Security</strong></p>
<ul>
<li>
<ul>
<li>Design and enforce security patterns for protecting data at rest and in transit across all Azure data platforms (e.g., Azure SQL, Cosmos DB, Databricks)</li>
<li>Partner with the Data Access Governance function to translate data classification policies into tangible technical access controls.</li>
<li>Architect solutions for secrets management, encryption, and key management, primarily leveraging Azure Key Vault.</li>
<li>Architect and design a secure data access solution using VDI (Azure Virtual Desktop), including the integration of Data Loss Prevention (DLP) and other data protection controls to prevent data exfiltration.</li>
</ul>
</li>
</ul>
<p style="padding-left: 40px;"><strong>Security Engineering &amp; Operations Enablement</strong></p>
<ul>
<li>
<ul>
<li>Develop security-as-code and Infrastructure as Code (IaC) to create guardrails and proactively detect insecure configurations.</li>
<li>Serve as a senior technical escalation point for the Security Operations team during complex cloud security investigations.</li>
<li>Evaluate, prototype, and recommend new cloud security technologies and services to mature the overall security program.</li>
</ul>
</li>
</ul>

<p style="padding-left: 40px;"><strong>Architectural &amp; Application Security Expertise</strong></p>

<ul>

<li>A firm understanding of core security principles like least privilege, defense-in-depth, and zero trust, and able to champion and educate colleagues on those principles.</li>
<li>Deep understanding of the OWASP Top 10, common application and cloud infrastructure security vulnerabilities, and their mitigation.</li>
<li>Hands-on experience with application security tools (SAST, DAST, SCA) and threat modeling methodologies such as STRIDE.</li>

</li>
</ul>

<li>Experience designing and implementing security controls for a variety of data platforms in Azure (e.g., Azure SQL, Cosmos DB).</li>
<li>Direct experience designing and securing VDI solutions in Azure (Azure Virtual Desktop), including network segmentation, endpoint security for images, and DLP policy enforcement.</li>
