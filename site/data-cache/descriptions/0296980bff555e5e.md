<p>At JetBrains, code is our passion. Ever since we started, back in 2000, we have strived to make the strongest, most effective developer tools on earth. By automating routine checks and corrections, our tools speed up production, freeing developers to grow, discover, and create.</p>
<p>We are looking for a Software Developer to join the core team developing <strong>Ktor</strong>, an asynchronous framework for building connected applications.</p>
<p>Ktor provides an HTTP server and client stack with multiple low-level subsystems (I/O, networking, protocols, etc.) and a rich set of extensible features. The team works on both evolving existing functionality and designing new framework capabilities.</p>
<h3><strong>In this role, you will:</strong></h3>
<ul>
<li>Work on the core of the Ktor framework.</li>
<li>Develop and improve HTTP server and client subsystems.</li>
<li>Design and implement new framework features and extensions.</li>
<li>Maintain and improve the existing codebase with a focus on quality, performance, and reliability.</li>
<li>Participate in design discussions and code reviews.</li>
<li>Collaborate closely with other developers and contributors.</li>
</ul>
