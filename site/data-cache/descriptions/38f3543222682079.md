<h3>The Role</h3>
We are looking for a <strong>Senior Site Reliability Engineer (SRE) to join the Compute Node</strong> team at Nebius AI Cloud. The Compute Node team is responsible for building and operating the cluster scheduler and node-level services that run and manage virtual machines across all cloud regions. This role focuses on Linux systems engineering, virtualization and operational reliability. You will work close to the operating system and hypervisor, shaping how reliability and observability are embedded into the Compute platform.</div>
<div><br><strong>Y</strong><strong>our responsibilities will include:</strong></div>
<ul data-list-tree="true" data-indent="0" data-border="0">
<li>Ensure reliability, availability and performance of compute nodes running VMs</li>
<li>Analyze and debug Linux systems across user space and kernel space, understanding capabilities, limitations and trade-offs at each layer</li>
<li>Troubleshoot complex production issues involving CPU, memory, NUMA, cgroups and scheduling</li>
<li>Work hands-on with virtualization and containerization, primarily using QEMU/KVM and Linux-native technologies</li>
<li>Design and evolve observability as a core capability of the node layer: metrics, logs, traces, alerts, SLIs and SLOs</li>
<li>Lead incident response, root-cause analysis, and postmortems, driving long-term reliability improvements</li>
<li>Collaborate closely with platform, kernel/hypervisor, GPU and infrastructure teams to improve system design and operability</li>
</ul>
