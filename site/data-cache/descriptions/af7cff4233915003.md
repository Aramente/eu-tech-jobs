You will bridge the gap between development and operations by embedding security directly into the CI/CD pipeline and managing complex identity and content delivery architectures, including <strong>headless AEM</strong> and <strong>Auth0</strong> integrations. Your role is pivotal in ensuring that our platform is not only high-performing but resilient against modern threats like <strong>DDoS attacks</strong>.</p>
<h3><strong>Key Responsibilities</strong></h3>
<ul>
<li><strong>Identity &amp; Access Management:</strong> Configure and manage <strong>Auth0</strong> integrations for AEM, including token validation, OIDC/SAML configurations, and custom login modules to ensure secure user authentication.</li>
<li><strong>Headless Security:</strong> Oversee the security of <strong>AEM Headless</strong> deployments, including protecting GraphQL endpoints, managing CORS policies, and ensuring secure communication for decoupled front-end frameworks (React/Angular).</li>
<li><strong>Edge &amp; CDN Protection:</strong> Manage and configure <strong>CDN</strong> (e.g., Cloudflare, Akamai, or Adobe-managed CDN) to optimize performance and implement <strong>DDoS mitigation</strong> strategies.</li>
<li><strong>Traffic Filtering:</strong> Implement and maintain <strong>Traffic Filter Rules</strong> and Web Application Firewall (<strong>WAF</strong>) configurations at the CDN level to block malicious spikes and sophisticated application-layer attacks.</li>
<li><strong>Automated Security Scanning:</strong> Integrate security tools (<strong>SAST/DAST</strong>) and secrets detection into <strong>CI/CD pipelines</strong> (Jenkins, GitLab) to identify vulnerabilities early in the development cycle.</li>
<li><strong>Environment Hardening:</strong> Install and manage AEM author, publish, and dispatcher instances with a focus on <strong>Dispatcher security hardening</strong>, SSL certificate automation, and ModSecurity configurations.</li>
<li><strong>Observability &amp; Incident Response:</strong> Monitor system performance and security logs using tools like Splunk or New Relic to proactively address threats and performance bottlenecks.</li>
<li><strong>Compliance Auditing:</strong> Regularly audit the platform and its integrations (Adobe Analytics, Target) to ensure alignment with corporate security policies and industry standards.</li>
</ul>
<h3><strong>Required Skills &amp; Experience</strong></h3>
<ul>
<li><strong>Experience:</strong> 5+ years in administering and securing <strong>AEM environments</strong>.</li>
<li><strong>Identity Services:</strong> Proven experience integrating <strong>Auth0</strong> or similar Identity Providers (IdP) for enterprise-scale authentication.</li>
<li><strong>Architectural Knowledge:</strong> Strong understanding of <strong>Headless CMS</strong> security best practices, including API key management and JWT authentication.</li>
<li><strong>Network Security:</strong> Expertise in managing <strong>CDNs</strong> and implementing <strong>DDoS mitigation</strong> and <strong>WAF</strong> rules.</li>
<li><strong>Technical Stack:</strong> Proficiency in <strong>Apache Sling, JCR, OSGi</strong>, and web servers like <strong>Nginx</strong> or <strong>Apache</strong>.</li>
<li><strong>Automation:</strong> Hands-on experience with scripting (Python) and CI/CD tools (Jenkins, CircleCI) to automate security and deployment workflows.</li>
<li><strong>Cloud Experience:</strong> Experience with cloud-based AEM implementations, such as <strong>AEM as a Cloud Service (AEMaaCS)</strong> or AWS/Azure.</li>
<li><strong>Diagnostic Skills:</strong> Proficiency in analyzing log files, thread dumps, and heap dumps to resolve security incidents or performance outages.</li>
</ul>
