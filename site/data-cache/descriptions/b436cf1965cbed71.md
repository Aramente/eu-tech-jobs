The Elastic Search AI Platform, used by more than 50% of the Fortune 500, brings together the precision of search and the intelligence of AI to enable everyone to accelerate the results that matter. By taking advantage of all structured and unstructured data — securing and protecting private information more effectively — Elastic’s complete, cloud-based solutions for search, security, and observability help organizations deliver on the promise of AI.</p></div><h3><strong>What is The Role</strong></h3>
<p>We are looking for a security-focused Engineer to strengthen Kibana's security posture and build robust platform security features. In this role, you'll be responsible for hardening Kibana against evolving threats, managing security vulnerabilities, and ensuring our application meets the highest security standards for enterprise and government customers. You'll also leverage AI to build innovative security tools that enhance our vulnerability detection, automate security workflows, and accelerate threat response. If you're passionate about web application security and want to make a direct impact on protecting critical infrastructure while pushing the boundaries of AI-powered security, this is the role for you.</p>
<h3><strong>What You Will Be Doing</strong></h3>
<ul>
<li>Lead security hardening efforts across Kibana's codebase and infrastructure, including content security policy implementation and enforcement</li>
<li>Build AI-powered tools and workflows to enhance security operations, including automated vulnerability detection, intelligent security alert triage, and predictive threat analysis</li>
<li>Manage third-party dependency security through regular audits, vulnerability assessments, and coordinated upgrades</li>
<li>Collaborate with security researchers and respond to vulnerability reports with urgency and thoroughness</li>
<li>Design and implement security controls for authentication, authorization, and auditing features</li>
<li>Work closely with Operations and Engineering teams to ensure security best practices across our hosted and on-premise offerings</li>
<li>Contribute to threat modeling and security architecture decisions for new features</li>
<li>Write secure, maintainable code for both client and server-side components</li>
</ul>
<h3>What You Bring</h3>
<ul>
<li>At least 3 years of web development experience, with a focus on secure development practices</li>
<li>Strong knowledge of web application security principles including OWASP Top 10, secure authentication patterns, and defense-in-depth strategies</li>
<li>Experience with security vulnerability management, including triage, remediation, and coordinated disclosure</li>
<li>Proficiency with JavaScript, TypeScript, and Node.js</li>
<li>Familiarity with security scanning tools and vulnerability management platforms (e.g., Snyk, CodeQL, HackerOne)</li>
<li>Ability to work and communicate effectively with a worldwide team in a distributed work environment</li>
<li>Hands-on experience implementing and maintaining content security policies, CORS policies, and other browser security controls</li>
<li>Understanding of single-sign-on technologies, role-based access control, and identity management protocols</li>
