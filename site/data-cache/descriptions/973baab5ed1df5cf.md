<p><strong>About The Team:</strong><br>This role oversees the smooth operation of the automated equipment during production; from setting up the equipment to analysing data to provide improvement areas. This is done through effective communication and using bespoke software tools. The Flow Controller mainly interacts with Operations Managers, Shift Manager, Engineering and Automation Tech teams. They monitor production lines, report issues, and help solve problems. They also assist with testing new systems and work on improving box production efficiency by addressing bottlenecks, stock issues, and quality problems, coordinating teams as needed.</p>
<p><strong>What You Will Be Doing</strong>:&nbsp;</p>
<ul>
<li>Use Warehouse Control Systems (WCS) and Production Control Systems (PCS) to ensure that machine statuses are active and correctly set up.</li>
<li>Identify and, when possible, anticipate conveyor jams. Notify engineering if needed and ensure balanced box flow.</li>
<li>Conduct regular checks on line throughput, analysing choke points, and taking remedial action to address any issues.</li>
<li>Effectively coordinate service and support functions to ensure the smooth operation of the facility.</li>
<li>Provide reports and analysis on machine performance</li>
</ul>
