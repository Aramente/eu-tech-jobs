<p>Make Wayve the experience that defines your career!&nbsp;&nbsp;</p></div><p><strong>The role</strong></p>
<p>This is a high-impact Fullstack Engineer opportunity to build and own a new internal web application that becomes the single go-to place for teams to browse computer vision models, run them on data subsets, and explore results.</p>
<p>You will partner directly with internal users to shape workflows and ship an MVP quickly, then iterate based on real feedback. As the first dedicated Fullstack hire in Corpus &amp; Enrichment, you will set the standard for maintainable, scalable UI engineering that accelerates model development and data labelling workflows across the business.</p>
<p>Key responsibilities:</p>
<ul>
<li>Lead discovery with internal stakeholders to define user workflows, UX, and requirements for Enrichment Shop</li>
<li>Build, ship, and iterate the Enrichment Shop product end-to-end using React/TypeScript and established backend web frameworks</li>
<li>Integrate the frontend with existing systems and APIs (including Databricks-backed workflows and internal services)</li>
<li>Incorporate user feedback into delivery planning and help define lightweight expectations/SLAs for an internal tool</li>
<li>Build for long-term maintainability by writing clear, testable code and enabling others to contribute and support the system</li>
</ul>

<h2><strong>About you</strong>&nbsp;&nbsp;</h2>
<p>In order to set you up for success as a Senior Fullstack Engineer at Wayve, we’re looking for the following skills and experience.&nbsp;&nbsp;</p>
<p><strong>Essential</strong></p>
<ul>
<li>Strong JavaScript and TypeScript experience, with React as the preferred frontend direction</li>
<li>Proven track record delivering a full web application, ideally owning work end-to-end in production</li>
<li>Experience integrating with existing backend systems and APIs, and working effectively in an established codebase</li>
<li>Able to operate independently, partnering directly with internal stakeholders and building pragmatic, maintainable, testable solutions</li>

</ul>
