**Role**

We are looking for a highly skilled UI/UX Artist to elevate the user interface of **[Hole.io.](https://play.google.com/store/apps/details?id=io.voodoo.holeio&hl=en)** You must have strong 2D art skills, a refined sense of visual polish, and a proven ability to define and implement a complete UI vision working closely with the Art Director. This role requires hands-on experience in Unity, from designing UI elements in Figma to setting up and integrating them in-engine. You will collaborate closely with the Art Director, Game Designers, and Developers to craft interfaces that are both visually compelling and functionally intuitive.

You will be responsible for building and maintaining a consistent UI style across the game and ensuring your designs enhance gameplay clarity and player satisfaction. Your role also includes creating the UI Style Bible and component libraries in figma, ensuring that design systems are scalable and cohesive.

- Establish a high quality, scalable UI, from early concepts to final implementation.
- Create and maintain a comprehensive UI Style Bible and reusable component libraries.
- Design high-quality 2D assets for menus, HUDs, buttons, icons, and other UI elements.
- Setup, integrate, and animate UI scenes in Unity.
- Collaborate with the Art Director to ensure consistency with the game's visual tone.
- Work with Game Designers to ensure UI supports gameplay and accessibility goals.
- Ensure technical feasibility and performance of UI assets on mobile devices.
- Iterate quickly based on team feedback and live user data.
