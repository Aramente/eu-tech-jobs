<h1><strong>What you’ll do</strong></h1>
<ul>
<li><strong>Mobile development expertise:</strong> You have solid experience building high-quality, cross-platform applications for <strong>iOS and Android</strong> using <strong>React Native</strong> and <strong>TypeScript</strong>.</li>
<li><strong>Native familiarity:</strong> While you live in JavaScript, experience with native code (<strong>Swift, Kotlin</strong>, or <strong>Objective-C</strong>) to bridge gaps or debug native modules is a significant plus.</li>
<li><strong>Form handling:</strong> You have experience building complex forms and implementing robust validation logic. You know how to make data entry easy for the user while ensuring data integrity.</li>
<li><strong>Store deployment:</strong> Experience managing the release lifecycle - including certificates, signing, and deploying to the <strong>Apple App Store</strong> and <strong>Google Play Store </strong>is a big plus.</li>
<li><strong>Styling proficiency:</strong> You have a knack for creating polished mobile interfaces. Experience with <strong>Styled Components</strong> is a plus, but ultimately, you know how to build layouts that look and feel great on any screen size.</li>
<li><strong>User-obsessed mindset:</strong> You don't just write code; you build products. You are passionate about <strong>mobile UX</strong>, gestures, and animations. Experience with <strong>Figma</strong> is a plus, helping you bridge the gap between design and engineering.</li>
<li><strong>Clean code philosophy:</strong> You care deeply about the quality of your craft. You strive for simplicity, strictly adhere to <strong>DRY (Don't Repeat Yourself)</strong> principles, and write code that is readable and maintainable.</li>
<li><strong>Collaborative spirit:</strong> You are a genuine team player who thrives in a collaborative environment. You communicate clearly, welcome code reviews, and enjoy solving problems with your peers.</li>
<li><strong>Testing culture:</strong> You believe that code isn't complete until it's verified. Experience with mobile testing frameworks is highly valued to ensure reliability across devices.</li>
<li><strong>Performance focus:</strong> You understand that a janky app is a bad UX. You keep performance in mind, optimising for smooth animations, fast launch times, and efficient memory usage.</li>
<li><strong>Smart tooling usage:</strong> You leverage the right tools for the job—including <strong>AI assistants</strong>—to be efficient, but you are transparent about their use. You verify every output, ensuring that you maintain full understanding and ownership of the code you ship.</li>
<li><strong>Industry interest:</strong> While not required, previous experience in <strong>Web3</strong>, <strong>blockchain</strong>, or <strong>fintech</strong> is considered a plus.</li>
</ul>
