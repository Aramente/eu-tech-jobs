**At NordPass, we’re making password security effortless.**

We believe logins should be safer and simpler. Join the team behind NordPass— a password manager trusted by 7M+ users and 10,000+ global companies to secure credentials, cards, and sensitive data.

**Your impact?** Helping millions secure their digital lives by creating smarter, safer ways to log in.

## Main Responsibilities

- Build, extend, and own security-facing backend services (breach detection, email masking, and related tooling) with a high bar for quality, performance, and correctness.
- Use AI-assisted workflows daily (code generation, test scaffolding, debugging, performance tuning) and continuously evaluate new tooling that can improve the squad's output.
- Help establish shared AI practices within the team: review patterns, prompt libraries, and guardrails so AI-generated code meets the same standard as hand-written code.
- Participate in system design and architectural decisions, advocating for solutions that are secure, scalable, and maintainable.
- Collaborate cross-functionally with Frontend, Android, iOS, and Backend engineers to deliver end-to-end features across all platforms.
- Take ownership of what you build, from design through production, and hold yourself accountable for uptime, reliability, and code quality.

## Core Requirements

- 3+ years of hands-on PHP development in production systems.
- Demonstrable AI-native workflow: you already use tools like Cursor, Copilot, Claude, or similar daily for code generation, refactoring, test creation, and debugging, and you take full ownership of every output.
- Solid grasp of system design. You know which patterns fit which problems and can articulate trade-offs clearly.
- Strong understanding of OOP, asynchronous programming, and design patterns.
- Comfortable with CI/CD pipelines, unit testing, and integration testing as non-negotiable parts of shipping.
- Clear, proactive communicator who works well in a cross-functional environment.

## Bonus Points For

- Experience with Go (we use it alongside PHP).
- Knowledge of cryptography or security engineering fundamentals.
- Contributions to internal tooling, developer experience, or engineering enablement.
- Experience building or integrating AI agents, LLM-powered features, or automated pipelines beyond personal productivity.
